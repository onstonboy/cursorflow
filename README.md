## CursorFlow

Minimal Cursor presets for faster project setup.

---

## What this repo is

This repository contains a pre-configured `.cursor` folder (rules, commands, and UI/UX presets) that you can reuse in your own projects.

---

## How to use

1. Clone or download this repository.
2. Copy the `.cursor` folder from the root of this repo.
3. Paste it into the **root folder** of your own Cursor project (same level as your source code).
4. Open that project in Cursor – the commands and rules from this `.cursor` folder will now be available.

That’s it. Your project now uses the same Cursor setup as this repo.
